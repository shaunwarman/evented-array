## Install

[npm][]:

```sh
npm install evented-array
```

[yarn][]:

```sh
yarn add evented-array
```


## Usage

```js
const arr = new EventedArray();

arr.events.on('item:pushed', element => {
  console.log(`item pushed ${element}`);
});

arr.push(10);
console.log(arr[0]);

> item pushed 10
> 10
```
